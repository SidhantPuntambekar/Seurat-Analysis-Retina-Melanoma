# Seurat-Analysis-Retina-Melanoma

Dataset sources:
Melanoma data: https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE137710 \n
Retinal data: https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE116106
